# SyntaxWear E-commerce

Um projeto de e-commerce simples e responsivo, focado em moda, desenvolvido para demonstrar o uso de HTML e CSS modernos.

## Funcionalidades

-   **Design Responsivo:** Layout que se adapta a diferentes tamanhos de tela (desktop e mobile).
-   **Navegação Intuitiva:** Menu de navegação claro e funcional.
-   **Exibição de Produtos:** Grade de produtos para visualização.

## Tecnologias Utilizadas

-   **HTML5:** Para a estrutura semântica do conteúdo.
-   **CSS3:** Para estilização, layout (Flexbox/Grid) e responsividade.

## Como Usar

Para rodar este projeto localmente:

1.  Clone o repositório:
    `git clone https://github.com/lays-araujooo15/ecommerce-syntaxwear.git`
2.  Navegue até a pasta do projeto:
    `cd ecommerce-syntaxwear`
3.  Abra o arquivo `index.html` no seu navegador de preferência.

## Estrutura do Projeto

-   `index.html`: A página principal do e-commerce.
-   `css/`: Contém todos os estilos CSS, organizados por componentes.
-   `images/`: Contém todas as imagens usadas no site.
